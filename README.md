# Siege Launcher
All in one launcher to play Older versions of Rainbow Six Siege without messing with Steam Depot Downloader or files. Simply download the launcher and launch the season you want !

## Why would I use Siege Launcher?
Playing old seasons of Rainbow Six Siege can be frustrating for some players who doesn't know how to use cmd or don't know what to do with files in order to get the game working properly. Using Siege Launcher will get you ready to play in seconds (almost) without having to mess with anything, just click on download the game, launch it and the app will patch the game on the go for you.

## Installation
Grab the latest .exe file in the [Release](https://github.com/Artaic-fr/siege-launcher/releases) section, to install the launcher and start playing.

## Tech Used
### App side
- [VueJS](https://github.com/vuejs/core)

- [ThrowbackLoader](https://github.com/lungu19/ThrowbackLoader)
